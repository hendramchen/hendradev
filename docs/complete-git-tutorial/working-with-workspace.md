# Working with the Workspace

- Three states model: working directory, staging area, repository
- Checking status (git status)
- Viewing changes (git diff, git diff --staged)
- Adding changes (git add — single file, paths, interactive -p)
- Committing (git commit — message best practices, -m, --amend)
- Ignoring files (.gitignore rules & examples)
