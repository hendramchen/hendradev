# Rewriting History and Advanced Local Ops

## Amend commits

contents of the last commit can be changed

## Interactive rebase -- reword, squash, reorder, drop

contents of multiple commits can be changed

## Reset vs Revert

contents of multiple commits can be undone

## Cherry-pick

contents of a specific commit can be applied to the current branch

## Stashing

contents of the working directory can be temporarily saved
