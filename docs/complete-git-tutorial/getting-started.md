# Getting Started

Sebelum mulai melacak atau melakukan komit kode, Anda perlu menginstal Git, mengonfigurasi identitas Anda, dan memahami cara membuat atau mengkloning repositori. Bagian ini akan memandu Anda langkah demi langkah.

## 2.1 Install Git (macOS / Linux / Windows)

Instalasi Git bergantung pada sistem operasi Anda. Setelah instalasi, Anda akan memverifikasi apakah Git berfungsi dengan benar.

### macOS Installation

Option 1: Install via Homebrew (recommended)

```
brew install git
```

Option 2: Install Xcode Command Line Tools

```
xcode-select --install
```

Option 3: Download from official website

Official Git installer:
[https://git-scm.com/download/mac](https://git-scm.com/download/mac)
